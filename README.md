# ik8s
k8s多级群管理平台

## 语言
Django 3.0  
Python 3.7  

## 功能列表:  
1.多集群导入  
2.webkubectl  引用https://github.com/KubeOperator/webkubectl  
3.应用管理（deployment/pod/job/cronjob/daemonset/statefulset）  
4.服务与路由管理  
5.配置管理  
6.存储管理  
7.pod日志查看/shell终端/事件查看  
8.集群监控
